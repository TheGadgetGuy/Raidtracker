Description

This mod parses Raid logs from CT_Raidtracker for WoW and inserts raids in bbdkp. When the member or the guild to which the member belongs does not exist it will be created in the memberlist table. then the memberdkp table record will be inserted or updated. and inserted in the raidattendees table. Loot will be entered in the items table.
If dkp price was entered in the Wow addon then this will price will be retained. this depends on the dkp system settings.

Requirements

  1. bbdkp 1.2.6 or higher
  2. phpbb3.0.10 or higher
  3. PHP 5.1.0 and Libxml 2.6.0 (simplexml usage)
  4. ftp and founder admin access on your phpbb installation.

Installation

Quick install guide

  1. Unzip the zip file into /store/mods/
  2. Launch automod, choose the install link. this will copy all the files, perform the necessary edits.
  3. Then surf to /install/index.php, and you will see the database installer. Launch the database installer. This will install the acp module, and clear the caches (template, theme, imagesets)
  4. Once installed, you will find the ACP module added under the Item section in bbdkp ACP.

Setup

  1. Raidtracker needs a correct setup of the Events. see the file /contrib/en_event_triggers.xml, paste it into the Event trigger import screen and it will create triggers, as well as the event end even the DKP pool if it is missing.
  2. Secondly you need to set up an "Own Raid" called "Trash mob", otherwise loot from trash will not be recorded.
  3. two example xml files are joined. from headcount and ct_raidtracker
